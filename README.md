# CommentSnap

Darmowy generator obrazka komentarza/odpowiedzi w stylu social mediów (Next.js + Tailwind).

## Instalacja

```bash
npm install
npm run dev
```

Aplikacja: `http://localhost:3000/create`

## Szybki test MVP

1. Zmień username/komentarz (i opcjonalnie dodaj avatar) w zakładce **Comment** -> preview aktualizuje się live.
2. Kliknij **Copy** -> komentarz trafia do schowka i pojawia się toast `Copied!`.
3. Kliknij **Download Comment** -> pobierany jest PNG tylko z preview (2x pixel ratio).
4. Wejdź w **History** -> widoczna lista zapisanych projektów; **Load** ładuje projekt z powrotem na `/create`.
